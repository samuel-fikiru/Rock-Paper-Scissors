# Rock-Paper-Scissors Game

A web-based Rock-Paper-Scissors game with autoplay, score tracking, and persistent data storage. Built to practice JavaScript, DOM manipulation, and browser storage techniques.

## Features
- Play Rock-Paper-Scissors against the computer  
- **Autoplay mode**: let the game play automatically  
- **Score tracking**: win, loss, and tie counts recorded  
- **Reset score**: clear all current stats easily  
- **Persistent data**: scores are saved even if you refresh the page using `localStorage`  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript 
- Browser `localStorage` for persistent score data
